1. open XAMPP, and start 'Apache' and 'MySQL' first.
2. open phpMyAdmin, create a database named 'kangaroocare_db'
3. inside 'kangaroocare_db' database, you will see 'Import' at the top navigation bar...
4. choose 'kangaroocare_db.sql' file (download from github) and Click 'Import' button (at left-side-bottom)
5. Done!!
